# Web Data Extraction using BeautifulSoup

This project demonstrates web data extraction and analysis using Python. 
It includes scraping structured HTML data, extracting URLs, checking website 
availability, and handling dynamic content using Selenium.

## Features
- Extracts headings, metadata, and HTML tags from webpages
- Collects and de-duplicates URLs from websites
- Identifies broken links using HTTP status codes
- Retrieves dynamic page source using Selenium
- Prepares scraped data for analysis and reporting

## Technologies Used
- Python
- BeautifulSoup
- Selenium
- Requests
- Pandas

## Use Case
Designed to support data collection and reporting tasks for digital and 
eCommerce platforms.
